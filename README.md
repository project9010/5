# Система электронного документооборота для АО "ОМЕГА"

Система управления документами на Python 3 и SQLite. Проект теперь содержит два интерфейса:

- консольную программу на Python;
- графическую оболочку на Java Swing, которая подключается к Python-бэкенду по HTTP.

Python-код остаётся бэкендом: он отвечает за базу данных, авторизацию, хеширование паролей и операции с документами. Java-код отвечает только за графическое окно пользователя.

## Структура проекта

```text
main.py          - консольная точка входа и навигация по меню
backend_api.py   - HTTP API на Python для подключения Java-клиента
OmegaDMSGui.java - графическая оболочка на Java Swing
auth.py          - регистрация, вход и SHA-256 хеширование паролей
documents.py     - создание, просмотр, поиск, удаление документов и изменение статуса
database.py      - подключение к SQLite и автоматическое создание таблиц
utils.py         - общие функции проверки ввода и вывода данных в консоль
```

## Возможности

- Регистрация и вход пользователей.
- Хеширование паролей SHA-256 с индивидуальной солью для каждого пользователя.
- Автоматическое создание SQLite базы данных при первом запуске.
- Создание, просмотр, удаление и поиск документов.
- Изменение статуса документа по фиксированному маршруту:
  - `created`
  - `in progress`
  - `approved`
  - `rejected`
- Защищённое выполнение SQL-запросов с параметрами для пользовательских данных.
- Консольный интерфейс на Python.
- Графический интерфейс на Java Swing.
- Связь Java и Python через локальный HTTP API: `http://127.0.0.1:8000/api`.

| Поле | Описание |
| --- | --- |
| `id` | Идентификатор документа |

## Как запустить консольную Python-версию

```bash
python main.py
```

## Как запустить Java GUI с Python-бэкендом

### 1. Запустите Python-бэкенд

В первом окне терминала выполните:

```bash
python backend_api.py
```

Если Python запускается командой `python3`, используйте:

```bash
python3 backend_api.py
```

После запуска появится сообщение:

```text
Python backend started: http://127.0.0.1:8000
```

### 2. Скомпилируйте Java-клиент

Во втором окне терминала выполните:

```bash
javac OmegaDMSGui.java
```

### 3. Запустите Java-клиент

```bash
java OmegaDMSGui
```

После этого откроется графическое окно. Через него можно зарегистрироваться, войти, создавать документы, искать документы, менять статус и удалять записи.

## Важное замечание

Для Java GUI сначала обязательно должен быть запущен `backend_api.py`. Если Python-бэкенд не запущен, Java-окно откроется, но регистрация, вход и работа с документами не будут выполняться.
